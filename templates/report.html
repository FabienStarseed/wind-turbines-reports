3. Inspection Methodology

  • UAV visual inspection performed on all three blade faces (LE, TE, PS, SS) per blade (A, B, C)
  • AI-assisted multi-stage analysis pipeline: triage (binary screening) → classification (defect typing + severity) → deep analysis (Cat 4–5 root cause)
  • Defect classification per Vestas Cat 1–5 severity scale, sourced from IEC 61400-23, DNVGL-ST-0376, and GWO Blade Repair Standard V5
  • Image processing: DJI P1 45MP images tiled at 1024×1024px with 20% overlap for AI analysis
  • Deep analysis for Cat 4–5 findings includes root cause, failure risk, Vestas SMP reference, and repair recommendation
