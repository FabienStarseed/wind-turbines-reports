**Add `load_and_resize_image()` utility function:**
```python
def load_and_resize_image(image_path: Path, max_edge: int = 1568) -> str:
    """Load image, resize if longest edge > max_edge, return raw base64 string (no data URI prefix)."""
    from PIL import Image
    import io
    with Image.open(image_path) as img:
        if img.mode != "RGB":
            img = img.convert("RGB")
        if max(img.size) > max_edge:
            img.thumbnail((max_edge, max_edge), Image.LANCZOS)
        buf = io.BytesIO()
        img.save(buf, format="JPEG", quality=85)
        return base64.b64encode(buf.getvalue()).decode("utf-8")
```

This function replaces the raw `open(image_path, "rb").read()` + `base64.b64encode()` pattern used in the old Gemini code. Using `thumbnail()` preserves aspect ratio. The 1568px limit is from Anthropic vision docs as the threshold where downscaling preserves full model performance.
