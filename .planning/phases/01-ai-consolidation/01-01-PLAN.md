```python
def call_claude_triage(
    image_b64_list: List[str],  # list of raw base64 strings (NO data:image/... prefix)
    blade: str,
    zone: str,
    position: str,
    turbine_id: str,
    turbine_model: str,

    location_type: str,  # "onshore" or "offshore"

```

Build content list: for each b64 in image_b64_list, append `{"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": b64}}` (CRITICAL: raw base64 only, no "data:image/jpeg;base64," prefix — this is the key difference from Kimi/OpenAI format).

Then append `{"type": "text", "text": user_prompt}` where user_prompt is TRIAGE_USER_PROMPT formatted with all context fields. Set `offshore_flag` to the offshore warning string when `location_type == "offshore"`, else empty string.

Retry loop for `MAX_RETRIES = 3` (configurable param):
- Catch `anthropic.RateLimitError`, `anthropic.OverloadedError`: sleep `(2 ** attempt) * 10` seconds (10s, 20s, 40s), retry. If last attempt, return `{"error": "rate_limit", "has_defect": True}` (conservative fallback — flag all remaining).
- Catch `anthropic.APIConnectionError`, `anthropic.APITimeoutError`: sleep `2 ** attempt` seconds (1s, 2s, 4s), retry. If last attempt, return `{"error": "connection", "has_defect": True}`.
- Catch `anthropic.APIStatusError as e`: if `e.status_code >= 500`, treat as transient (sleep, retry). If 4xx (client error, e.g. bad key), raise immediately — don't retry.
- Catch `json.JSONDecodeError`: per decision, return `{"has_defect": True, "confidence": 1.0, "error": "json_parse_fail"}` — do NOT retry.
- On success: strip markdown fences (same regex as existing code), parse JSON, return dict with added `"input_tokens": response.usage.input_tokens, "output_tokens": response.usage.output_tokens`.

Call: `client.messages.create(model=model, max_tokens=300, system=TRIAGE_SYSTEM_PROMPT, messages=[{"role": "user", "content": content}])`. Access response text via `response.content[0].text.strip()`.
