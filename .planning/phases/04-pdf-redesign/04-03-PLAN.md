**2. Defect page renderer — `_render_defect_page(pdf, defect, defect_index, total_defects)`:**
- Per CONTEXT.md: 1 defect per page, spacious layout, grid layout, large 80x80mm thumbnails, full metadata
- Layout (top to bottom):
  - Page title: "Defect Finding {defect_index}/{total_defects}" in BRAND_NAVY bold 14pt
  - Severity colour band: full-width coloured rect with category label (use SEVERITY_COLORS_IEC keyed by defect["category"])
  - Two-column area:
    - Left column: 80x80mm defect image via `_embed_defect_image()` using defect["image_path"]
    - Right column: metadata fields as label:value pairs
      - Defect ID, Defect Name (bold)
      - IEC Category + urgency
      - Zone / Position
      - Size estimate
      - Confidence score (as percentage)
      - NDT Recommended (Yes/No)
  - Visual description: multi_cell full width, italic
  - If defect["analysis"] is not None:
    - "Deep Analysis" sub-heading
    - Root cause, Recommended action, Repair timeframe, Estimated cost
    - Failure risk: safety risk level
    - Engineer review badge if required

**3. Action matrix renderer — `_render_action_matrix(pdf, report_data)`:**
- New page with "Action Matrix" heading
- Table with columns: Priority | Blade | Defect ID | Defect Name | Category | Zone | Timeframe | Action
- Each row colour-coded by severity (use SEVERITY_COLORS_IEC for fill, with readable text colour from text_rgb)
- Sorted by priority (P1 first — already sorted in report_data["action_matrix"])
- Use fpdf2 manual cells with `set_fill_color()` + `cell(fill=True)` for colour bands
- Handle page breaks: check `will_page_break(row_height)` before each row
- If no defects: show "No defects found — turbine in good condition" message

**4. Blade map renderer — `_render_blade_map(pdf, blade_label, blade_defects)`:**
- Per CONTEXT.md: programmatic drawing with fpdf2 shapes, mini-icons on diagram, severity colour on markers
- Draw a blade schematic as a zone grid:
  - Rows: LE, TE, PS, SS (4 zones)
  - Columns: Root, Mid, Tip (3 positions)
  - Each cell: 30x15mm rectangle
  - Cell fill colour = worst defect severity in that zone+position (use SEVERITY_COLORS_IEC). If no defects in cell, use light grey (230, 230, 230).
  - Zone/position labels on cells
- Blade label heading: "Blade {blade_label}" bold 12pt
- Zone labels on left margin, position labels across top
- After grid: severity legend showing all 5 IEC categories with colour swatches
- Helper: `_worst_cat_in_zone(defects, zone, position)` — find highest category defect matching zone+position, return int or -1 if none
- Defect count annotation: number of defects per cell shown as small circle with count
